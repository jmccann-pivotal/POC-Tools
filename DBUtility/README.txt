Utility files for POC use:

analyze_all.sh - Analyze all tables in a database one-by-one

bloat.sh - Vacuum Analyze all catalog tables in a database to reduce bloat

check_roles_resque.sql - See which users are in which resource queues

format_load_logs.sh - Reads the output of a series of INSERT/SELECT from
external tables scripts and generates CSV file of the tables and number
of rows inserted.

list_sequences.sql - list all active sequences in a database

lock_status.sql - check status of locks in a database

make_orafuncs.sh - install the Oracle functions in a database

resource_queue_monitor.sql - generic SQL commands to check resource queues
